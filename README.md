# Sentinel

Sentinel is an AI-powered UI QA agent. A headless browser collects **objective
evidence** — a screenshot, the accessibility tree, and action results — and
Claude judges whether the screen matches a natural-language **spec**.

It is built for the conditions that are painful to cover with conventional
end-to-end tests: database connection errors, network failures, request
timeouts, API errors, auth/session errors, and other abnormal or error states —
while remaining usable for ordinary spec verification too.

> **Status:** early development (pre-MVP). APIs, configuration, and behavior are
> unstable and may change without notice.
